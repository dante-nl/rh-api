How to cretae a simple updater

Hi! Welcome to the first article, about a simple updater! Note: This is meant to be an embed updater

Written by dante_nl, on July, 23.


Start by adding a "Get contents of URL" action to a new shortcut, or an existin one on the place where you want the updater

Now you need to fill in this URL: https://routinehub.co/api/v1/shortcuts/[RH ID]/versions/latest and replace [RH ID} with your Shortcut's RoutineHub ID. If you run it now, you should see something like this:

{"result": "success", "id": 00000, "Version": "1.0", "URL": "https://www.icloud.com/shortcuts/AABba0", "Notes": "- Example", "Release": "July 23, 2020"}

If you just open the iCloud URL, you don't get an extra download, so we use another way. Use an "Get dictionary value" action to get the value from the key "id".
